# Kurokami Backend

## Prerequisites
- Node.js
- MongoDB running locally (default port 27017)

## Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Server
- **Development Mode** (with auto-restart):
  ```bash
  npm run dev
  ```
  The server will run on `http://localhost:5000`.

- **Production Start**:
  ```bash
  npm start
  ```

## API Endpoints
- `GET /health`: Health check
